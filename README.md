## 💿 Installation et Utilisation

Une version **BÊTA** publique est prévue pour le **10 juillet**. Elle permettra de jouer au jeu du début à la fin en français.

**Procédure (à la sortie du patch) :**
1. Téléchargez le patch `.xdelta` depuis l'onglet **Releases** de GitHub.
2. Utilisez un logiciel comme **xdelta UI** ou **DeltaPatcher**.
3. Sélectionnez votre ISO d'origine (`ULES01557`) et appliquez le patch.
4. Lancez le jeu sur un émulateur (**PPSSPP**) ou une PSP modifiée.
